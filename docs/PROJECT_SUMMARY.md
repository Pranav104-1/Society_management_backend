## 🚀 How to Run

### Step 1: Install Dependencies
```bash
# Backend dependencies
npm install

# Frontend dependencies
cd frontend
npm install
```

### Step 2: Start Services
```bash
# Terminal 1 - Backend (from root)
npm run dev

# Terminal 2 - Frontend (from frontend folder)
cd frontend
npm run dev
```

### Step 3: Open Browser
```
http://localhost:5173
```

## 🔄 State Management

### Zustand Stores Implemented

#### useAuthStore
```
- user: Current user object
- isLoggedIn: Boolean flag
- loading: Loading state
- error: Error message
- setUser(): Update user
- logout(): Clear user
```

#### useComplaintStore
```
- complaints: Array of complaints
- loading: Loading state
- error: Error message
- setComplaints(): Update list
- addComplaint(): Add new complaint
```
